### How To local RUN

You can using **launch.json** in your visual studio code

1. use `go mod tidy`
2. use `go mod download`
3. use `go mod vendor`
4. use command `go run main.go`

You can using ```docker``` and ``` docker compose ```
```
docker-compose up --build -d
```
